# Assignment 3: Functions (User-Defined, Recursive, Lambda, Map, Filter)

This assignment demonstrates small reusable Python functions to automate basic tasks such as calculations, filtering products, and generating summaries.

## Contents
* **`task-1.py`**: Basic Function (Price After Discount)
* **`task-2.py`**: Recursive Function (Factorial Utility)
* **`task-3.py`**: Lambda Function (GST Calculator)
* **`task-4.py`**: Using `map()` (Apply GST to List of Prices)
* **`task-5.py`**: Using `filter()` (Filter Expensive Products)
* **`task-6.py`**: Combined Utility Function (`map()` and `filter()`)
* **`task-7.py`**: Mini Problem (Menu Using Functions)

## How to execute
Run each task directly from the command line:
```bash
python task-1.py
```
*(and so on for the others)*
