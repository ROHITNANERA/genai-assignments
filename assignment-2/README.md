# Assignment 2: Python - Control Flow (Conditionals & Loops)

This repository contains tasks demonstrating basic control flow mechanisms in Python, adhering strictly to constraints that require sticking to logical statements and loops without using functions or advanced patterns.

## Concept Breakdown

- **task-1.py:** Evaluates string input, conditionally handles incorrect input types, and routes numeric inputs through standard `if/elif/else` ladders to resolve layered business logic (discounting).
- **task-2.py:** Applies a uniform `for` loop to step through sequence data and compute running aggregates without manually redefining conditional steps inside.
- **task-3.py:** Provides a basic state machine using a `while True` loop interacting with standard input, routing user selection logic via conditionals, and showing the effect of `continue` and `break`.
- **task-5.py:** Actively manipulates loop execution using `break` and `continue` keywords to handle anomalies in a data sequence efficiently.

## How to Run

Launch your terminal or command prompt and run any of the task files independently using the Python interpreter:

```bash
python task-1.py
```
*(Replace `task-1.py` with the appropriate file name for each individual task).*

Each file includes an example output block at the bottom to verify correct behavior.
