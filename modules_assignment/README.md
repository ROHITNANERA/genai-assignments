# Assignment 5: Importing, Creating Modules & Packages

This assignment focuses on securely mapping python scopes utilizing modules (`math_utils.py`, `string_utils.py`) dynamically packaged sequentially through `shop_package/`.

## Contents
* **`math_utils.py`**: Mathematical function endpoints utilizing external scopes.
* **`string_utils.py`**: A generic text editor mapping text modification boundaries.
* **`shop_package/`**:
  * **`discount.py`**: Evaluates float price tags and offsets static sums.
  * **`billing.py`**: Evaluates full totals recursively taxing subsets securely.
  * **`__init__.py`**: Dynamically bundles child python endpoints locally natively.
* **`main.py`**: Iterative test executable dynamically querying all sub-packages at once securely mapping scopes properly into memory.

## How to execute
Everything exists within local relative folders meaning no virtual environment installations are needed whatsoever.
```bash
python main.py
```
